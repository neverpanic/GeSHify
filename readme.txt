GeSHify
-------

Thanks for downloading GeSHify. To install GeSHify, extract the archive in your Expression Engine directory, upload the added files and enable GeSHify in the Extension Manager in your control panel by going to Admin » Utilities » Extensions Manager and clicking “enable” in the table row for GeSHify.
If you're updating from a previous version of GeSHify the process is the same.
Please report any bugs you encounter or your feature wish list to neverpanic@gmail.com.
